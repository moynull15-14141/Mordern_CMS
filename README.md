# SportingSpy Enterprise Publishing Platform

This repository contains the foundation for a next-generation AI-powered enterprise publishing platform.

## Structure

- `apps/` - deployable applications
- `packages/` - shared reusable packages
- `config/` - environment and tool configuration
- `docker/` - container definitions
- `.github/` - CI/CD workflows

## Getting started

1. Install dependencies: `pnpm install`
2. Run development server: `pnpm dev`
3. Run backend: `pnpm dev:backend`
4. Run admin: `pnpm dev:admin`

## Phases

- Phase 1: project initialization
- Phase 2: backend foundation
- Phase 3: frontend foundation
- Phase 4: admin panel

## Notes

Follow the documentation in `project document/` for architecture and implementation planning.
